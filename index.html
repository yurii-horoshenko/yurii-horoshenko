Building VisiTime

A booking marketplace for barbershops, beauty salons and service businesses. Two apps (Client + Business) shipped on iOS and Android from a single KMM core, backed by an ASP.NET Core 9 API.

  • Two client apps on iOS (8 build targets: Live/Preview/Stage/Demo × 2)
  • Two Android apps sharing a common Compose design system
  • KMM shared module — single source of truth for business logic
  • ASP.NET Core 9 + PostgreSQL + Dapper, deployed via Azure Pipelines
  • AI agent factory built on Claude Code — spec → code → review → QA
  • Linear workflow driven by agents (PM → Architect → Dev → QA)
